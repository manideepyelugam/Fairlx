# Utility Scripts

Shell scripts for development and maintenance tasks.

## Scripts

### `fix-cookie.sh`
Fixes cookie-related issues during development.

**Usage:**
```bash
bash scripts/utils/fix-cookie.sh
```

### `run-usage-test.sh`
Runs usage billing system tests.

**Usage:**
```bash
bash scripts/utils/run-usage-test.sh
```

## Making Scripts Executable

```bash
chmod +x scripts/utils/*.sh
```
